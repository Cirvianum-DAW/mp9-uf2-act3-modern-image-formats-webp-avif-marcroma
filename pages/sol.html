What format is the most optimal for this type of image and why? With what compression quality does the image not lose quality? Can we reduce its size even more without losing quality? What would happen if we save the image several times with JPEG format?

The most optimal format for these kind of imagery is the JPEG format. Since the image is a photograph, it is best to use a format that is designed to handle photographs. If we save the image several times with JPEG format, the image will lose quality each time it is saved, because JPEG is a lossy format, which means that it loses quality each time it is saved.
